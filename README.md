# 🔐 Password Strength Checker

This is a simple password strength checker I built using HTML, CSS, and JavaScript.  
It checks how strong a password is based on a few rules and gives instant feedback with a percentage + visual indicator.

---

## 💡 What it does

When you type a password, the app looks at things like:

- how long it is  
- whether it has uppercase and lowercase letters  
- numbers and special characters  
- repeated characters  
- simple patterns like `abc` or `123`

Then it gives you a score from 0% to 100% so you can quickly see how strong your password is.

---

## ⚙️ How it works

It’s basically a scoring system:
- strong passwords get points  
- weak patterns lose points  
- everything gets combined into one final score  

At the end, the score is shown with a color so you can instantly understand it.
